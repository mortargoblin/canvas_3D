# canvas_3D

![F-15](/img/F15.png)
![car](/img/car.png)

Renders wireframes for 3D-models. Made with javascript.
[hosted here](https://canvas-3d.netlify.app/)

## specs

- 2-color wireframe projection
- supports obj loading
- supports arbitrary model position and angle updates

## self-hosting

### with obj loading

1. clone it
2. launch a server (e.g python -m http.server)
3. open localhost:8000 (or whichever port is used) with a browser

### without obj loading (default cube)

1. clone it
2. open index.html with a browser
